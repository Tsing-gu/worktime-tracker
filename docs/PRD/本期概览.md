# 本期概览 PRD

> 版本: v1.0  
> 日期: 2026-07-15  
> 状态: 待实现

## 1. 背景

当前「本周概览」按自然周（周一至周日）统计工时，无法反映节假日前后的真实工作节奏。例如国庆前一周可能只有3个工作日，但统计仍按完整周展示。

改为以**连续非工作日**为分界，统计两个"休息段"之间的工作日段（即"本期"），更贴合实际工作节奏。

## 2. 核心概念

| 概念 | 定义 |
|------|------|
| 非工作日 | 放假日（holidays 表 is_off_day=1）+ 不调休的周六日 |
| 调休补班日 | holidays 表中 is_off_day=0 的记录，属于工作日 |
| 本期 | 两个连续非工作日段之间的工作日区间 |
| 本期起点 | 上一个连续非工作日段结束后第一个工作日 |
| 本期终点 | 下一个连续非工作日段开始前最后一个工作日 |
| 总工作天数 | 本期起点到终点的全部工作日数（含调休补班日，排除周六日和法定假期）|
| 已工作天数 | 本期起点到今天中有上班记录且非请假的工作日数 |
| 剩余天数 | 今天之后到本期终点之间的工作日数 |
| 目标工时 | 总工作天数 × daily_required |
| 日均 | 已工作工时 / 已工作天数（未来日期不计入）|
| 假期加班 | 假期中的工时记录，不计入本期统计 |

## 3. 区间计算算法

### 3.1 判断某天是否为非工作日

```
def is_rest_day(dt, holidays):
    1. 查 holidays 表，is_off_day=1 → True（放假日）
    2. 查 holidays 表，is_off_day=0 → False（调休补班日，是工作日）
    3. 无记录 → dt.weekday() >= 5 → True（普通周六日）
```

### 3.2 计算本期区间

```
def get_period_range(today, holidays):
    1. 如果今天本身是非工作日 → 返回 None（休息中）
    2. 从今天向前搜索，找到最近的连续非工作日段
       - 连续 = 前一天和当天都是 is_rest_day
       - 段 = 连续非工作日的集合（至少2天）
    3. 从今天向后搜索，找到最近的连续非工作日段
    4. 本期起点 = 上一段最后一个非工作日 + 1天
    5. 本期终点 = 下一段第一个非工作日 - 1天
    6. 边界：
       - 如果向前搜索 365 天无连续非工作日段 → 起点 = 当年1月1日
       - 如果向后搜索 365 天无连续非工作日段 → 终点 = 当年12月31日
```

### 3.3 统计计算

```
def get_period_stats(today, records, holidays, daily_required):
    period = get_period_range(today, holidays)
    if period is None:
        return PeriodStats(is_rest=True)  # 显示「休息中」

    total_period_workdays = 0  # 整个本期所有工作日数（用于目标）
    worked_days = 0            # 有上班记录的工作日数
    worked_hours = 0.0         # 已工作总时长

    # 遍历整个本期区间
    d = period.start
    while d <= period.end:
        if is_workday(d, holidays):
            total_period_workdays += 1
            if d <= today:
                rec = records[d]
                if rec and rec.start_time and not on_leave:
                    worked_days += 1
                    if rec.end_time:
                        worked_hours += rec.total_hours
                    else:
                        worked_hours += now - start  # 实时计算
        d += 1

    remaining_days = count_workdays(today+1, period.end, holidays)
    target = total_period_workdays * daily_required
    remaining_needed = max(0, target - worked_hours)
    remaining_per_day = remaining_needed / remaining_days if remaining_days > 0 else 0

    daily_avg = worked_hours / worked_days if worked_days > 0 else 0
```

## 4. 界面展示

### 4.1 正常工作日

| 行 | 内容 | 示例 |
|----|------|------|
| line1 | `已工作 {worked_days}天 / {total_period_workdays}天` | 已工作 15天 / 20天 |
| line2 | `累计 {worked_hours}h / 目标 {target}h` | 累计 156.5h / 目标 168h |
| line3 | `日均 {daily_avg}h, 剩余{remaining_days}天 每天需{remaining_per_day}h达标` | 日均 7.8h, 剩余5天 每天需8.1h达标 |
| line4 | 区间日期 `{start}~{end}` | 6.20~9.24 |
| 进度条 | worked_hours / target | |

### 4.2 本期最后一天（remaining_days=0）

line3 改为：`日均 {daily_avg}h，今天已经是最后一天了，还剩{remaining_needed}h可下班`

### 4.3 休息日（周六日 / 法定假期中）

| 行 | 内容 |
|----|------|
| line1 | `休息中` |
| line2 | 空 |
| line3 | 空 |
| line4 | 空 |
| 进度条 | 0 |

## 5. 边界情况

| 场景 | 处理 |
|------|------|
| 今天是普通周六日 | 显示「休息中」|
| 今天在法定假期中 | 显示「休息中」|
| 假期加班工时 | 不计入本期统计 |
| 调休补班日 | 正常计入工作日 |
| 年初无前一连续非工作日段 | 起点 = 当年1月1日 |
| 年末无后一连续非工作日段 | 终点 = 当年12月31日 |
| 本期只有1天 | 正常统计 |
| 今天是本期最后一天 | line3 显示「日均xx.xh，今天已经是最后一天了，还剩xx.xh可下班」|
