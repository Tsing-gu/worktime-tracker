# 本月概览 PRD

> 版本: v1.0  
> 日期: 2026-07-15  
> 状态: 已实现

## 1. 背景

原「本月概览」使用自然日口径（含周末），只统计到今天，且未考虑请假。改为工作日口径，与「本期概览」格式一致。

## 2. 核心概念

| 概念 | 定义 |
|------|------|
| 本月范围 | 自然月 1日 ~ 月末 |
| 工作日 | 周一至周五 + 调休补班日，排除周六日和法定假期 |
| 请假天 | 从总工作天数中扣除（不算应工作、不算已工作、目标也减少）|
| 总工作天数 | 本月范围内的工作日数（排除请假天）|
| 已工作天数 | 到今天为止有上班记录且非请假的工作日数 |
| 剩余天数 | 今天之后到月末的工作日数（排除请假天）|
| 目标工时 | 总工作天数 × daily_required |
| 日均 | 已工作工时 / 已工作天数 |

## 3. 界面展示

| 行 | 内容 | 示例 |
|----|------|------|
| line1 | `已工作 {worked_days}天 / {total_workdays}天` | 已工作 10天 / 22天 |
| line2 | `累计 {worked_hours}h / 目标 {target_hours}h` | 累计 92.3h / 目标 176h |
| line3 | `日均 {daily_avg}h, 剩余{remaining_days}天 每天需{remaining_per_day}h达标` | 日均 9.2h, 剩余10天 每天需8.4h达标 |
| line4 | 隐藏 | |
| 进度条 | worked_hours / target_hours | |

### 本月最后一天（remaining_days=0）

line3 改为：`日均 {daily_avg}h，今天已经是最后一天了，还剩{remaining_needed}h可下班`

## 4. 涉及文件

| 文件 | 改动 |
|------|------|
| `src/core/calculator.py` | `get_month_stats()` 改为返回 `PeriodStats`，工作日口径 + 请假处理 |
| `src/services/worktime_service.py` | `get_month_stats()` 返回类型改为 `PeriodStats` |
| `src/ui/main_window.py` | 本月卡片数据填充改为与本期概览一致格式，line4 隐藏 |

## 5. 验证方式

1. 验证周六日和法定假期不计入总工作天数
2. 验证调休补班日计入工作日
3. 验证请假天从分母扣除
4. 验证剩余天数 = 今天之后到月末的工作日数（排除请假天）
5. 验证目标 = 总工作天数（排除请假）× daily_required
6. 验证 line4 隐藏
